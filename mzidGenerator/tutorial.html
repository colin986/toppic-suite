Installation

Linux

    sudo apt-get install python3 #only if Python3 is not installed already
    sudo apt-get install python3-pip #only if pip3 is not installed already

    pip3 install lxml
    pip3 install six
    pip3 install sqlalchemy==1.3.24
    pip3 install numpy
    pip3 install pyteomics

Windows

  1. Download the latest version of python from Python website if not installed already.

  2. Enter following commands to install required libraries.

    pip3 install lxml
    pip3 install six
    pip3 install sqlalchemy==1.3.24
    pip3 install numpy
    pip3 install pyteomics

Command line usage

Command format

python3 write_mzIdent.py <path-to-prsm-tsv-file> <path-to-fasta-or-fasta_target_decoy-file> <optional: path-to-fixed-ptm-file> <optional: path-to-common-PTM-file> <optional: path-to-variable-PTM-file>

Examples

Generate mzIdentML file from a PrSM identification file by TopPIC prsmId.tsv and a protein database file database.fasta.

  python3 write_mzIdent.py prsmId.tsv database.fasta None None None

Generate mzIdentML file from a PrSM identification file by TopPIC prsmId.tsv and a protein database file database.fasta. During proteoform identification, a ptm file fixed_mods.txt was used for fixed modifications.

  python3 write_mzIdent.py prsmId.tsv database.fasta fixed_mods.txt None None

Generate mzIdentML file from a PrSM identification file by TopPIC prsmId.tsv and a protein database file database.fasta. During proteoform identification, a ptm file common_mods.txt was used for PTM localization and a ptm file variable_mods.txt was used for variable PTM modifications.

  python3 write_mzIdent.py prsmId.tsv database.fasta None common_mods.txt variable_mods.txt
